# NPainter
Simple Raster-based Painter written in Nim (Work in Progress)

## Building
Only works on Linux/X11. Arch/Manjaro is Recommended
```nim
# Build Binary and Pack SVG Icons
nimble build -d:packIcons
# Build Binary Only
nimble build
```
